# MVC Design Patterns & RESTful API Interview Prep

### 16. Explain the MVC architectural pattern and its importance in Node.js applications.
**Explanation:** Scalable design structure.
**Answer:**  
MVC separates Models (data), Views (presentation), and Controllers (request handling), enabling organized, testable, and maintainable code.

---

### 17. What is REST? What makes an API RESTful?
**Explanation:** API design conventions for networked applications.
**Answer:**  
RESTful APIs are stateless, use resource-oriented URLs, standard HTTP verbs, and meaningful response codes for predictable client-server interactions.
